뉴스정보원연결망 (News Source Networks, NSN)

뉴스정보원연결망(News Source Networks, NSN)이란 관심있는 뉴스 집합에서 동시에 뉴스에 등장한 정보원들을 연결한 네트워크로서, 해당 키워드의 중심 인물 또는 단체 등의 정보원을 시각적으로 확인할 수 있고 주변 정보원들을 네트워크로 연결하여 인물 또는 단체와의 관계를 확인할 수 있음

여기서 한 뉴스의 정보원이란, 해당 뉴스의 원천(Source)이 되는 정보를 발언한 인물 또는 단체를 일컫고, 이때의 주요 정보는 따옴표(“”)를 기준으로 기사에 나타난 인용문으로 정의함

또한 뉴스의 주요 정보인 인용문의 감정을 분석(Sentiment Analysis)하여 해당 키워드에 대한 정보원의 입장 또는 극성(긍정, 중립, 부정)을 결정 짓고, 뉴스정보원연결망을 통해 정보원들의 극성을 시각적으로 표현함

한 정보원의 한 키워드에 대한 극성을 결정하는 방법은, 해당 정보원이 특정 키워드에 대해 발언한 문장의 모든 감정 분석 결과 중 개수가 많은 극성으로 결정 지으며, 이때 가장 강한 극성을 나타내는 문장을 “가장 극단적인 발언”이라고 간주하여 해당 정보원을 대표하는 문장으로 추출하여 요약함

프로젝트 개요 (Project Overveiw)

뉴스정보원연결망 프로젝트는 크게 데이터 수집, 정보원 및 인용문 추출, 인용문에 대한 감정 분석, 네트워크 구축으로 나뉨

1. 데이터 수집

  • 매체: 15개의 언론사
    종합지: 중앙일보, 동아일보, 국민일보, 내일신문, 서울신문, 세계일보, 한겨례, 한국일보
    경제지: 매일경제, 서울경제, 파이낸셜 뉴스, 헤럴드 경제
    방송사: MBC, SBS
    전문지: 전자신문
  • 기간: 2017.01.01 ~ 2017.12.31
  • 키워드: 10개
    소년법, 가상화폐, 적폐청산, 법인세, 부동산, 정규직, 남북관계, 거시경제, 원자력발전소, 최저임금
  • 건수: 약 9만 건의 뉴스 기사

2. 정보원 및 인용문 추출

  • 인용문 추출: 큰 따옴표(“”) 기준 인용문 추출
  • 정보원 추출: 큰 따옴표 앞의 말뭉치에서 정보원 추출
    Python KoNLPy의 MeCab 형태소 분석기 사용 (은전한닢 프로젝트)
    (http://konlpy-ko.readthedocs.io/ko/v0.4.4/)
    (https://bitbucket.org/eunjeon/mecab-ko-dic)
  • 정보원 판별: 말뭉치의 이중 파싱과 전처리 이후 가중치를 부여하여 정보원 판별

3. 인용문 감정 분석: KOSAC 감정 사전 사용 (http://word.snu.ac.kr/kosac/)

4. 네트워크 구축

  • 네트워크 객체: Python Networkx API (https://networkx.github.io/)
  • 네트워크 시각화: d3.js 기반 netjsongraph.js (https://github.com/netjson/netjsongraph.js)
